# Instagram MERN Project

Welcome to the Instagram MERN Project repository! This project aims to replicate some of the core functionalities of the popular social media platform Instagram using the MERN (MongoDB, Express.js, React, Node.js) stack.

## Features

- User Authentication: Sign up, log in, log out functionality with JWT authentication.
- Post Creation: Users can create posts with captions and images.
- Like and Comment: Users can like and comment on posts.
- Profile Management: Users can edit their profiles, including profile picture and bio.

## Technologies Used

- MongoDB: NoSQL database used for storing user data, posts, comments, etc.
- Express.js: Backend framework for building RESTful APIs.
- React: Frontend library for building user interfaces.
- Node.js: JavaScript runtime environment for running server-side code.
- JWT: JSON Web Tokens used for authentication.
- Mongoose: MongoDB object modeling tool for Node.js.
- Axios: Promise-based HTTP client for making requests from the frontend to the backend.
- Material-UI: React UI framework for building responsive and attractive user interfaces.

## Getting Started

To get started with the project, follow these steps:

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Install dependencies by running `npm install` in both the `client` and `server` directories.
4. Create a `.env` file in the `server` directory and set up environment variables (such as MongoDB URI, JWT secret, etc.).
5. Start the backend server by running `npm start` in the `server` directory.
6. Start the frontend development server by running `npm start` in the `client` directory.
7. Access the application in your browser at `http://localhost:3000`.
